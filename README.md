
 Music Playlist Manager

A console-based Java project using OOP to manage music playlists.

 Features
- Add new songs to the playlist
- Remove songs from the playlist
- Search for a song by title
- Display all songs
- Optional: Sort songs alphabetically
- Optional: Save and load playlists from a file

Classes
- Song: Represents a single song (title, artist, album, duration)
- Playlist Manages a list of songs
- Main: Menu-driven console program for user interaction
